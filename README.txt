How to run the tests:
1, Start the selenium hub with the following command in the terminal:
java -jar src/test/resources/selenium-server-4.27.0.jar hub

2, Start the selenium nodes with the following command in the terminal:
 java -jar src/test/resources/selenium-server-4.27.0.jar node

3, Right click on the src/test/resources/SmokeTestRunner.xml file and run it
